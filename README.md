📘 physics-simulation — 物理挙動シミュレーションアプリ
🎯 目的（Purpose）
このアプリは 物理挙動の理解と可視化を目的として開発されました。
ユーザーが 基本的な物理現象（衝突・レイキャストなど）を直感的に学習できることを重視しています。

🧩 構造（Structure）
アプリは以下の最小構造で動作します：

マウス入力（Input）  
オブジェクトの選択・移動・操作など

処理（Logic）

レイキャストアルゴリズム

衝突判定（Collision Detection）

物理演算（PyBulletによるシミュレーション）

出力（Output）

画面描画

必要に応じてファイル保存（画像・ログなど）

状態管理（State）

オブジェクトの位置・速度

シミュレーションの進行状態

入力状態の保持

UI / UX

シンプルな操作

学習を妨げない最小UI

物理挙動が見やすい画面構成

構造は 普遍的な物理モデルを理解しやすいよう最小化されています。

🛠 使用技術（Tech Stack）
言語：Python

テキストエディタ：kusor

物理エンジン：PyBullet

開発思想：最小構造・高速試行・教育向けシンプルUI

📦 機能（Features）
ボックス操作機能  
マウスでオブジェクトを掴む・動かす・落とすなどの操作が可能

衝突シミュレーション  
オブジェクト同士の衝突をリアルタイムで可視化

レイキャスト可視化  
マウス位置からのレイを飛ばし、衝突点を表示

物理パラメータ調整（任意）  
重力・摩擦・反発係数などを変更可能な構造に拡張しやすい

🚧 例外・エラー処理（Exceptions）
アプリでは以下の例外を局所化して処理しています：

衝突計算時の不正値

レイキャスト対象が存在しない場合

入力が無効な場合

例外処理は メイン構造に影響しないよう分離されており、
学習体験を妨げない設計です。
